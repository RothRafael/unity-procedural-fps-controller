# First Person Controller – Unity

A customizable **First Person Controller** built in Unity.

---

## Features

- Smooth movement  
- Sprint  
- Crouch  
- Jump  
- Zoom  
- Procedural camera effects (headbob, noise)  
- Sound and UI decoupled using Events  

---

## Controls

| Key        | Action               |
|------------|----------------------|
| **WASD**   | Movement             |
| **Space**  | Jump                 |
| **Shift**  | Sprint               |
| **CTRL**   | Crouch               |
| **Z**      | Zoom                 |
| **H**      | Show / Hide UI       |
| **ESC**    | Lock / Unlock Mouse  |

---

## Setup

1. Clone the repository  
2. Open the project in Unity  
3. Open the main scene  
4. Press **Play**
