# First Person Controller – Unity

A customizable First Person Controller built in Unity.

## Demo

Playable demo:  
https://rothrafael.itch.io/first-person-controller-demo

## Controls

- **WASD** – Movement  
- **Space** – Jump  
- **Shift** – Sprint  
- **CTRL** – Crouch  
- **Z** – Zoom  
- **H** – Show/Hide UI  
- **ESC** – Lock/Unlock Mouse  

## Setup

1. Clone the repository  
2. Open the project in Unity  
3. Open the main scene  
4. Press Play
